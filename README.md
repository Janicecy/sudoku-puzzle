# Sudoku-puzzle
[Sudoku](https://janicecy.github.io/sudoku-puzzle/) - Watch your computer play Sudoku.
## Built With
- React + TypeScript
- Material-UI
## Setup
Install dependencies:
```
yarn install
yarn start
```
