# Sudoku-puzzle
[Sudoku](https://janicecy.github.io/sudoku-puzzle/) - Watch your computer play Sudoku. Visualization of how backtracking algorithm can solve a sudoku puzzle.
## Built With
- React + TypeScript
- Material-UI
## Setup
Install dependencies:
```
yarn install
yarn start
```

## Features 
- ✅ Auto-play
- ✅ Select difficulty 
- ✅ Erase Previous step 
- ✅ Highlight selected cell
